Welcome to the Internet Topology Zoo. The Zoo is an ongoing project to collect data network topologies from around the world.


We currently have over two hundred and fifty networks in the Zoo, available in a variety of graph formats suitable for statistical analysis, plotting, or other network research.

The below image shows the coverage of the zoo - each dot represents a node in a network.


The network graphs can be found in the dataset, or as plotted maps in the gallery. We also provide a set of tools for converting these to other formats.


The networks are manually traced from operator provided network maps. For more information on the process please see the documentation section.


We are always on the look out for more networks to add to the zoo. If you can provide a link to a network map we don't have, please let us know via the contribute page.


Other collections of graphs exist on the Internet. Some can be found here.


We hope you are able to make use of the networks in the Zoo. If you do, please let us know. We also ask that you cite us appropriately in any publications.
